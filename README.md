## The "Must-Haves" (Core Scientific Functions)

#### These are the non-negotiable functions that define a scientific calculator:

1- **Trigonometry**: $sin$, $cos$, $tan$ and their inverses ($sin^{-1}$, $cos^{-1}$, $tan^{-1}$).  

2- **Logarithms**: $log$ (base 10) and $ln$ (natural log, base $e$).

3- **Powers & Roots**: Square root ($\sqrt{x}$), $x^2$, $x^y$ (universal power), and $\sqrt[y]{x}$ (nth root).

4- **Constants**: Quick-access buttons for $\pi$ and $e$. 

5- **Scientific Notation**: An EXP or EE button for entering numbers like $6.022 \times 10^{23}$.

6- **Parentheses**: Essential for handling the Order of Operations (PEMDAS/BODMA
